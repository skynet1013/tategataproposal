# Vertical Drama AI Planner (縦型ドラマAIプランナー)

Vertical Drama AI Planner is a SaaS tool designed to help creators generate viral scripts, storyboards, and concepts for vertical short dramas (TikTok, Reels, YouTube Shorts) in seconds using AI.

## Features

- **Rapid Planning**: Generate detailed scripts and storyboards from simple inputs (genre, target audience).
- **Viral Optimization**: Optimize content for specific goals like "High Retention", "Save Count", or "Like Count".
- **Visual Storyboard**: AI-generated image previews for each scene (Mock/Prototype).
- **History Management**: Save and manage your past proposals.
- **PDF Export**: Export your plans for sharing with production teams.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com) v4
- **UI Components**: [Shadcn UI](https://ui.shadcn.com) (Radix UI + Tailwind)
- **State/Form**: React Hook Form + Zod
- **AI Integration**: Mock Service (Phase 1) / OpenAI & Anthropic (Planned Phase 2)

## Getting Started

1. Install dependencies:
   ```bash
   cd web
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment

This project is optimized for deployment on [Vercel](https://vercel.com).

1. Push to GitHub.
2. Import project in Vercel.
3. Set **Root Directory** to `web`.
4. Deploy.
